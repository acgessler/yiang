Highscore  About

YIANG is a self-contained Jump'n'Run game featuring an unique ASCII look. It offers 32 levels with dozens


of different enemies, small brainteasers and hidden scoring possibilities. The game is played for high score and gets faster and faster the longer you survive.

It's fun. All you need is your keyboard. See the floating screenshots for a few impressions, or take a look at this page's background which shows some of the game's easier levels (don't want to scare you off).

Download YIANG (Windows 2000, XP, Vista, 7) - 9.8 MiB, full source code included.

No installation needed. Simply download the archive, extract it somewhere on your hard disk and run one of the run_... files in the main folder (i.e. run run_fullscreen.bat to put the game in fullscreen mode).

Linux/Mac

Of course, YIANG works on Linux as well. It's a bit more complicated to run, however, and not suitable for minors and cowards. Please download the full game from the download link above and follow the instructions in the INSTALL file.

Mac users basically need the same procedure, but slight changes might be necessary ... ;-)
